What are the different ways to select an item?

You can select an item by any element tag. For example an ID, class, or simply p, or h.

What is the difference between an ID and a Class?

Class selectors can be used in many instances, whether ID can only be used once.

What is the difference between a SPAN tage and a DIV tag?

SPAN is for inline alterations only, DIV is for block elements.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV tags have no significant purpose, while HEADER, MAIN, FOOTER serve an intentional structural meaning.

What are the different ways to select a color?

You can simply type the name of the color for a predetermined system color. For more percision, you can use hex colors, or RGB(A) for maximum customization.
